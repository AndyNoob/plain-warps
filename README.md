# Plain Warps
Just warps, that's it. 
## Installation
Find latest build in https://github.com/AndyNoob/plain-warps/actions and drag and drop into your `/plugins/`.
## Usage
### `/warp <warp_name>` (player)
### `/warps add|remove <warp_name> (pos xyz) (rot)` (admin)
